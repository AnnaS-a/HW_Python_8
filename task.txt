Задача.
Создать телефонный справочник с возможностью импорта и экспорта данных
в файле на одной строке хранится одна часть записи, пустая строка разделитель
в файле на одной строке хранятся все записи, символ разделитель ","

Есть модуль логирования, который заносит в справочник всю поступающую информацию .
В системе есть модуль, выполняющий обращения для получения данных и 
модуль генерации html-представления.
Запуск системы осуществляется из головного модуля.

Необходимо:
модуль 1: сбор информации от пользователя    input_data.py
    в нем создадим модули, позволяющие собрать данные
    add_contact(data, columns)         (пользователь вводит данные, при создании контакта)
    def input_column()       (пользователь определяет название столбца при его создании) 
    
модуль 2: логирование                   logger.py 
    в нем создадим модули, позволяющие логировать собранные данные
    (разнообразные функции для работы приложения) и 
    указываем наименование файла куда все записывается 
    write_contact(user, data)      (запись контакта)
    write_data(data, columns)           (запись данных в файл) 
    add_column(data, column, columns)     (добавление столбца) 
    read_data()       (чтение файла)
    def get_columns(data)             (создание столбца)
    find(data)             (поиск по справочнику)
    del_contact(data, column)       (удаление столбца с его значением)

модуль 3: Запись в файл   inform_log.py
    модуль записи  write_log() 

модуль 4: главный модуль                inf_base.py
    отсюда запускаем сиситему
